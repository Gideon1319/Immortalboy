"Operators"

An operator is a symbol that represents an operation that may be performed on one or more operands.
    Types :
      1. Unary Operators.
      2. Binary Operators
    1. Unary operators : It operate on only one opreand and provide their single operand.


    2. Binary Operator : The operators operate on two operands.
      i. Arithmetic operator.
      ii. Relational (comparision) operator.
      iii. Assignment operator.
      iv. Logical operator.
      v. Bitwise operator.
      vi. Special operator.
        ---->Membership.
        ---->Identify.
    ====> i. Arithmetic Operator :

    Arithmetic operator are used to perform operator like addition, subtraction, multiplication.


    ====> ii. Relational Operator :

    Evaluate the compare value an either side of the expression and it returns true or false.


    ====> iii. Assignment Operator :

    It is used to assign the values of expression of the variables.


    ====> iv. Logical Operator :

    It combaines two or more condition and perform the logical operations using the logical, and logical, or logical, not logical.


    ====> v. Bitwise Operator :

    Bitwise operators works with bits of a binary number it operators bit by bit.


    ====> vi. Special Operator :

    Membership : It is used to test whether a value or variable is found is a sequance string,list,tuple,set dictionary is not in are the membership operators.


    E.g :
      >>>x="Hello World"
      >>>y={1,a,2,b}
      >>>print('W' in x)
        True
        >>>print('n not in x')
          True
    Identity Operator : Identity Operator are used to determine whether the value of variable is of a certain tuple or not, is and, is not are the identity operators.


    E.g :
      X1=5
      Y2=10
      print(X1 is Y2)
        False
      print(X1 is not Y2)
        True
