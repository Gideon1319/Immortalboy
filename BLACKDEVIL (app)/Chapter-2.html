Statements :
*A statement is an instruction that the python interpreter can execute.
*There are two kinds of statements : assignment (=) and print.

    1. Assignment statement :
An assignment statement assoxiates the name to the left of the '=' symbol with the object denoted by the expression to the right of the '=' symbol.
    E.g:
    >>>Name='xxxx'
    >>>Age=9
After an assignment, the values stored in the variable 'Name' and 'Age' are remembered.
    E.g:
    >>>Name
      xxxxx

    >>>Age+2
      11
    2. Simultaneous Assignment statement :
*Python permits any number of variables to appear on the left side, seperated by commas.
*The same number of expressions must then appear on the right side, again separated by commas.
    E.g:
      >>>x=10
      >>>y=5
      >>>sum,diff,prod=x+y,x-y,x*y
      >>>sum
        15
      >>>diff
        5
      >>>prod
        50
    3. Print statement :
*The print statement takes a series of values separated by commas.
*Each value is converted into a string (by implicitly invoking the string function) and then printed.
(E.g) :

      enter your name : yyyy
